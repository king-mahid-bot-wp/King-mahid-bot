## 🌟 About Mahid WhatsApp Bot

> **Mahid WhatsApp Bot** is a powerful, feature-rich **Multi-Device WhatsApp Bot** built with Node.js. It supports group management, AI chat, media downloads, games, and much more — all from your WhatsApp! Created and maintained by **King Mahid**.

## ⚙️ Configuration

### 🔑 Permission Levels

```js
permission: 0  →  All users (public)
permission: 2  →  Bot administrators only
permission: 3  →  Bot administrators + Group administrators
```

### 🏷️ Prefix Modes

```js
prefix: true    →  Only with prefix (e.g./ /help)
prefix: false   →  Without prefix (e.g./ help)
prefix: 'both'  →  Both modes work
```

### 🌍 Default Config

```json
{
  "PREFIX": "/",
  "timeZone": "Asia/Dhaka",
  "commandMode": "public",
  "autoUpdate": true
}
```

### 1️⃣ Fork the Repository

<div align="center">
  <a href="https://github.com/https://github.com/king-mahid-bot-wp/King-mahid-bot/fork">

</div>

### 2️⃣ Clone & Install

```bash
# Clone the repository
git clone https://github.com/king-mahid-bot-wp/King-mahid-bot.git
# Go into the folder
cd king-mahid-bot

# Install dependencies
npm install
```

### 3️⃣ Run the Bot

```bash
node index.js
```
